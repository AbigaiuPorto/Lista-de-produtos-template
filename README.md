# Lista-de-produtos-template

Projeto Loja Virtual

# Descrição

Este projeto consiste em uma página web que simula uma loja virtual, exibindo uma lista de produtos com imagem, nome, preço, parcelamento e avaliação.

Os produtos são gerados dinamicamente utilizando JavaScript, sem a necessidade de criar os elementos diretamente no HTML.

# Tecnologias utilizadas

HTML5
CSS3
JavaScript

# Funcionalidades

Exibição de produtos em formato de cards
Imagens dos produtos
Preço formatado em Real (R$)
Parcelamento em até 10x
Avaliação com estrelas
